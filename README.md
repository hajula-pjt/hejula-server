   2. 숙소 검색

   1. 대시보드 
      1. 이번 달 예상 총 방문자 수

      5. 숙박 현황
   2. 숙소 분석
      1. 방문객 연령대
      2. 방문객 태그
      3. 명칭
      4. 월별 숙소 방문객 수
